Hexo&&Github在Mac上搭建自己的微博

发表于 2017-09-22

搭建流程

在github上搭建hexo博客大概流程如下：

  • 搭建环境准备（node.js和git环境，github账户）
  • 安装以及配置Hexo
  • 将Hexo和github联系起来
  • 发布新博客
  • 更改主题

环境配置

Node.js

用来生成静态页面。详情Node.js官网，一路安装即可。

Git

用来将本地Hexo内容提交到Github上。可以参考Hexo官网上安装方法（Xcode中自带Git）。

安装以及配置Hexo

Node.js和Git都安装好的前提下即可正式安装Hexo了，打开终端执行如下命令：

$ sudo npm install -g hexo

终端cd到一个选定好的目录，进行初始化：

$ hexo init blog

blog是你建立的文件夹名称。cd到blog文件夹下，安装npm：

$ npm install

安装好npm后，开启hexo服务器：

$ hexo s

在浏览器中访问地址http://localhost:4000,能看到如下页面：

注意：在配置所有的_config.yml文件时（包括theme中的），在所有的冒号:后边都要加一个空格，否则执行hexo命令会报错。

在blog目录下执行生成静态页面命令：

$ hexo generate 或者 hexo g

再执行配置命令：

$ hexo deploy 或者 hexo d

注意：若执行命令hexo deploy仍然报错：无法连接git或找不到git，则执行如下命令来安装hexo-deployer-git：

$ npm install hexo-deployer-git --save

然后从新执行hexo g和hexo d命令（或者hexo d -g）。

发布新博客

终端进入到blog目录下，执行如下命令新建文章：

$ hexo new "postName"

名为postName.md的文件会建立在目录/blog/source/_posts下。你可以用相关markdown编辑器进行编辑。

文章编辑完成后，终端进入到blog目录下，执行如下命令：

hexo generate

hexo deploy

或者 hexo d -g。

成功后，即可在网上浏览新发布的文章了。

更换Theme

你可以到Hexo官网主题页去搜寻自己喜欢的theme。这里以hexo-theme-next为例

终端进入blog目录，执行如下命令

$ git clone https://github.com/iissnan/hexo-theme-next themes/next

打开_config.yml文件编辑theme对应的名称landscape为next

然后执行如下命令（每次部署文章的步骤）：

$ hexo clean //清楚缓存文件（db.json）和已经生成的静态文件（public）

$ hexo g //生成缓存和静态文件

$ hexo d //部署到服务器

至于更改theme内容，比如名称，描述，头像等去修改blog/_config.yml文件和blog/themes/next/_config.yml文件中对应的属性名称即可， 不要忘记冒号:后加空格。 NexT 使用文档里有极详细的介绍。

补充

添加ssh key到Github

####步骤一：检查SSH Keys是否存在Github

执行如下命令，检查SSH keys是否存在。如果有文件id_rsa.pub或id_dsa.pub，则直接进入步骤三将SSH key添加到Github中，否则进入下一步生成SSH key。

$ ls -al ~/.ssh

步骤二：生成新的ssh key

执行如下命令生成public/private rsa key pair，注意将your_email@example.com换成你自己注册Github的邮箱地址。

$ ssh-keygen -t rsa -C "your_email@example.com

默认会在相应路径下（~/.ssh/id_rsa.pub）生成id_rsa和id_rsa.pub两个文件。

步骤三：将ssh key添加到Github中

Find前往文件夹~/.ssh/id_rsa.pub打开id_rsa.pub文件，里面的信息即为SSH key，将这些信息复制到Github的Add SSH key页面即可。

进入Github --> Settings --> SSH keys --> add SSH key:

Title里任意添一个标题，将复制的内容粘贴到Key里，点击下方Add key绿色按钮即可。
